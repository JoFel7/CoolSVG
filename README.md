
# CoolSVG

## Description
Generates a simple svg logo based on inputs provided by the user.


## Demo

https://drive.google.com/file/d/1CTuJXVOmV1Yll-uyhTXAcF3hzWtg_OmI/view?usp=drive_link


## Installation

Install my-project with npm

```bash
  npm i
  node index.js
```
    
## Test With Jest
```bash
  npm test
```
